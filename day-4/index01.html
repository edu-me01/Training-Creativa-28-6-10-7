Day 4: JavaScript Examples

DOM Manipulation – Change Content

Use getElementById to select and update an element's text content.

const heading = document.getElementById('title');
heading.textContent = 'Welcome Back!';

DOM Manipulation – Add Event Listener

Add an event listener to respond to button clicks.

document.getElementById('btn').addEventListener('click', () => {
  alert('Button Clicked!');
});

Arrays – Basic Access

Accessing elements using array indexing.

const fruits = ['Apple', 'Banana', 'Mango'];
console.log(fruits[1]); // 'Banana'

Arrays – Push Method

Add an item to the array using push.

const tasks = [];
tasks.push('Study JavaScript');
console.log(tasks);

Array Modification – Insert with Splice

Use splice to insert elements at a specific index.

const colors = ['red', 'blue'];
colors.splice(1, 0, 'yellow');

Loops – forEach

Iterate over arrays using forEach with access to index.

const tasks = ['Design', 'Code', 'Test'];
tasks.forEach((t, i) => console.log(`${i + 1}: ${t}`));

localStorage – Save & Get

Store a string in localStorage and retrieve it.

localStorage.setItem('theme', 'dark');
console.log(localStorage.getItem('theme'));

localStorage – Store Objects

Use JSON.stringify to store objects.

const user = { name: 'Ali', level: 3 };
localStorage.setItem('user', JSON.stringify(user));

Mini Project – Task Stats

Calculate basic task statistics.

function getStats(tasks) {
  return {
    total: tasks.length,
    done: tasks.filter(t => t.done).length,
    pending: tasks.filter(t => !t.done).length,
  };
}
